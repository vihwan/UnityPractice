# 개발 일지

## 확인된 버그
* UI 상태창이 해상도에 맞게 조절되지 않는다.
* 가끔 지형의 고도에 따라 캡슐이 지멋대로 돌아간다.
* 왼쪽 마우스를 클릭하여 공격하면, 공격 모션이 여러번 실행된다.
* 특정 무기를 든 상태에서 똑같은 무기로 교체할 경우, 교체 모션이 나온다.
